Sube el archivo a wp-content/plugin2. Activar plugin3. Ir a Apariencia > widgets4. Arrastra el widget al sidebar que quieras.5. Comprueba que el versiculo diario se ve correctamente!    - Widget: Se puede añadir un widget de Últimas Noticias en la barra lateral de tu sitio de wordpress, vaya a apariencia - widget "Últimas Noticias"   - Template: en cualquier lugar de su plantilla inserte este codigo `<script type="text/javascript" language="javascript" src="https://www.bibliatodo.com/assets/js/wordpress/es/widget-news.js"></script>` . == Screenshots ==1.
